# Olá, eu sou o Moises! 👋

## 💻 Sobre mim

- 🎓 Cursando **Análise e Desenvolvimento de Sistemas (ADS)**.
- ⚛️ Especialista em **React.js** e desenvolvimento de componentes reutilizáveis.
- 🛡️ Uso de **TypeScript** para garantir código seguro, robusto e de fácil manutenção.
- 📐 Domínio de layouts complexos e responsivos com **Flexbox** e **CSS Grid**.
